## ğŸš€ Getting Started

### Quick Start

Choose your preferred tech stack and navigate to the corresponding directory:

```bash
# Frontend Options
cd frontend/react-app       # React
cd frontend/nextjs-app      # Next.js
cd frontend/html-vanilla    # Vanilla HTML

# Backend Options
cd backend/django-api       # Django
cd backend/flask-api        # Flask
cd backend/node-express-api # Node/Express
```

Follow the README in each directory for specific setup instructions.

### Full Stack Setup

1. Choose a frontend option
2. Choose a backend option
3. Follow deployment guides in `cloud/`
4. Configure environment variables
5. Run the application
